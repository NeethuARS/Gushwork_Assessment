# Gushwork Frontend Assignment

## Overview
Responsive product page built using vanilla HTML, CSS, and JavaScript.

## Features Implemented
- Sticky header (appears after first fold)
- Sticky price header (appears after price section)
- Image carousel with arrows + thumbnails
- Image zoom on hover
- Fully responsive layout (desktop, tablet, mobile)
- Multiple carousels (applications, process, testimonials)
- Modal interactions (quote + specs)

## Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript

## Notes
- Pixel-accurate implementation based on Figma
- No external libraries used
- Optimized for responsiveness and performance
